Test program for Z80 hardware emulator  
https://github.com/hasegawa-tomoki/z80-hardware-emulator

# Requirements

zasm, asembler for the Zilog Z80 cpu.  
https://k1.spdns.de/Develop/Projects/zasm/Distributions/

# Usage

```
$ zasm prelim.z80
```
